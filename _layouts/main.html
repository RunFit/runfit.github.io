  • Run Fresh

    Run Fresh is a set of proven exercises and active stretching that strengthens muscles in your pelvis, legs, and feet. Running stresses the large muscle groups or prime movers in your legs - quadriceps, hamstrings, and calves. While these muscles get stronger, the smaller stabilizer muscles that support them get weak. This imbalance in strength can lead to injury. These exercises improve balance, strength, and agility.

    Download

  • Run Strong

    Run Strong is a set of six high intesity body weight exercises just for runners. These exercises performed with little rest push you to the limit increasing both upper and lower body strenth. Being a strong runner isn't just about having strong legs. Runners with strong upper bodies have more edurance, more power through hills, and better conditioning.

    Download

  • Run Ready

    Run Ready is a collection of time based running workouts for all the major distances. Traditional running plans direct you to run a fixed number of miles each day. However, this can lead to an obsession with pace and burnout. Volume and intensity are what really matter. Run Ready workout plans provide beginning, intermiediate, and advanced workouts for all levels of experience. Optional virtual coaching is also included.

About Run Fit?

Run Fit is a collection of running apps developed by Ely Greenberg and Erg Process Energy. Ely is a passionate runner, who's completed 12 marathons. He's developed these exercises over a decade of running. During this time he has managed to continuously run hard and remain injury free.
